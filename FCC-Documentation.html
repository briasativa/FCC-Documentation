Introduction

Once you have your HTML the way you want it, now what? The nest step to realizing your new website is to format the words the way you imagined for your finished site. When writing a document in Word or Google Docs, or another word processor, you can size your text, change color, bold or underline, even number your document to present the information as needed. With minor adjustments to your HTML, you can do the same with CSS. If you have been writing HTML, you may already know some CSS. Here we will demystify the text manipulation all in one place.

Using Selectors

There are a few ways to style the text. There are more complicated or combined ways to select styles but we're sticking to the basics for this document. The main ways are

  • Inline
  • Style Block
  • Element Selectors
  • ID Selectors
  • Class Selectors

You've may have already been been changing the style of the text or elements with inline selectors or using a style block by adding color: red; inside your elememt like:

<p color:red;>Hello World!</p>

It should look like (Hello World!).

The style block selector is CSS syntax in a 'style' element in HTML.

<style> p { color: red; } </style>

Using an element selector is similar to the style block but in CSS, instead of in the HTML. Just like in the style block, it makes it easy to style all of an element on the website at once. If you style a body element, all body elements have the same style, if you style all h2 elements, they all change, etc.

body { color: red; }

If more flexibility is needed the ID selector works on individual elements. Add an ID selectorto a single paragraph, heading, div, or any other element. The ID selectorcan only be used once but it makes custom changes easy. By adding an individual identity to an element it gets its own CSS style. A unique ID is added to an element, like this:

<p ID="red-text">Hello World!</p>

Then a "#"(pound) sign is used to connect the CSS to the element. It's important to use an ID that describes the style, for future reference and to avoid mix-ups.

#red-text { color: red; }

The last selector we'll use is the class selector. It is similar to the ID selector but it can be duplicated. For instance, if every second paragraph should be red, the same class selector can be used for each one. The class selector is useful if there needs to be a repeated style throughout the document.

<p class="red-text">Lorem ipsum dolor sit amet. Tempor incididunt ut labore.</p> <p>Consectetur adipiscing elit, sed do eiusmod.</p> <p class="red-text">Tempor incididunt ut labore et dolore magna aliqua.</p>

Then a "."(period) sign is used to connect the CSS to the element. It's helpful to use a class that describes what it does to reduce confusion.

.red-text { color: red; }

It would be simple to turn a ID into a class if you decide you want to repeat the style more often.

Using these selectors makes styling easy.
Changing Text Color
There are three main ways to change color in CSS using the above selectors. There are variations on these we will not cover.
  • Color keyword - like "red"
  • HEX value - like "#ff0000"
  • RGB value - like "rgb(255,0,0)"
  • HSL value - like "hsl(320, 100%, 30%)"
The colors are vast, so using a resource is important. You can use a search engine to find colors. Here are a couple resources that may be helpful. A HEX value reference is useful. And here's a color picker or reference. In the picker you can see the value of HEX, RGB, keywords and HSL and convert one to another. Many photo editors allow you to try different colors and copy the color value from the editor. It's a good way to play around with colors before you commit.
Text Fonts

Fonts can make a huge difference in the feel of your website. A font can communicate seriousness, informality, fun, ect. Changing fonts is simple. You can change the font in all the text in the body, or individual sections to make a point.

Many fonts are already available, like Arial, or Times. There is an abundance of fonts at Google fonts. They have the HTML links and the CSS for font families to make it easy. It is a good practice to have an alternative font down the font tree in case a specifice font doesn't work on other browsers.

If you are using a common font like, Helvetica, you can add the font directly to the CSS of the body or section you want to change.

body {font-family: 'Quicksand', sans-serif;}

If you want to play it safe or you know you want to use a unique font, the link to the font must be added to the HTML: <link href="https://fonts.googleapis.com/css2?family=Quicksand&display=swap" rel="stylesheet"> You could just add the

body {font-family: 'Quicksand';} but it's good to have a second, or even a third choice from the font family just in case. If you are unsure try web fonts for reference.

Emphasising Words in HTML
Inline text semantic can come in many forms, depending on the point you are trying to make. We are just going to cover what in copy writing would be bold, italics, and underline, how they are expressed, and why. While many emphasised words are to organize and improve visuals, word emphasis is not only for aesthetics. Readers for the sight impaired rely on some semantics to communicate what is important on a web page.

  • To make text bold, use the font-weight selector. font-weight: bold; Font weight 'normal' is the default. (font-weight: normal;) You can also choose a custom weight in multiples of 100, up to 900. (font-weight: 900;)
  • Italics are often used as captions, to emphasis a change in mood, or to point out a technical term.
  • Another way to emphasise text is underlining the text that needs to stand out.

In HTML emphasis can also help with accessability. Underline emphasis has some uses that can be confusing or unhelpful, so check out The Unarticulated Annotation (Underline) element
